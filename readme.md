Poiza dotfiles
--------------

*ready to stow!*

Install/Link a package: stow <package_name> (run inside the ~/dotfiles directory).
Remove a package: stow -D <package_name> (deletes the symlinks).
Re-link a package: stow -R <package_name> (removes and re-creates symlinks).
Adopt existing files: stow --adopt <package_name> (moves existing files in the home directory to your stow folder, replacing them with a symlink).
